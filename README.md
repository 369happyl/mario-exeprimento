# 🍄 Mario Jump Game

Um mini jogo inspirado no Mario feito com HTML, CSS e JavaScript.

## 🎮 Como funciona

- O Mario pula obstáculos
- O cano se move automaticamente
- Se bater no cano = game over
- Sistema de animação usando CSS
- Lógica usando JavaScript

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript

## 🧠 O que aprendi

- Manipulação do DOM
- addEventListener
- animações com CSS
- lógica de colisão
- position absolute
- @keyframes
- funções no JavaScript

## 📱 Feito no celular

Esse projeto foi desenvolvido praticamente inteiro pelo celular usando:
- WebCode
- GitHub Pages
